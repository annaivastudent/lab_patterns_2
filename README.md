### 1. Observer (Shape)
Каждая фигура хранит список слушателей.  
При изменении фигуры вызывается `notifyChange()`.

Используется в `Warehouse`, чтобы обновлять кеш площади, периметра и т. д.

### 2. Repository
Хранилище фигур.  
Позволяет:

- добавлять фигуры
- удалять
- искать по ID
- фильтровать с помощью спецификаций
- сортировать с помощью компараторов

### 3. Specification
Паттерн для фильтрации.  
Реализованы спецификации:

- `ByIdSpec`
- `ByNameSpec`
- `QuadrantSpec`
- `AreaRangeSpec`
- `DistanceRangeSpec`

Любая спецификация поддерживает **оба метода**:

- `isSatisfied(obj)`
- `isSatisfiedBy(obj)`

### 4. Warehouse
Один-единственный объект (Singleton), который:

- кеширует площадь, периметр и объём фигуры
- автоматически сбрасывает кеш при изменении фигуры (Observer)
